# 2026-06-02

## 요약 (AI 작성)

zntc 번들러/트랜스파일러를 대대적으로 다진 날입니다. 깊은 좌결합 이항 체인과 prefix 타입 연산자(keyof keyof…)에서 터지던 스택 오버플로우를 없애려 파서·트랜스포머·코드젠·private-mangler의 재귀 AST walker를 반복 순회로 바꾸고(#4123 PR-1~2c), 재귀 walker 재유입을 막는 CI tripwire까지 세웠습니다. 번들러는 cross-chunk 전역 네이밍으로 동명 export collapse·namespace re-export 충돌(#4101)과 CJS default/named interop(#4120)을 바로잡고, fan-out discovery dispatch를 batch-drain으로 amortize(#4003 회귀)했습니다. 여기에 cold/incremental 빌드 벤치마크 인프라(zntc vs esbuild vs rolldown)를 붙이고 GitHub Pages 차트로 실측을 공개했습니다. suji는 os-info·autostart 플러그인과 suji.config.ts 빌드 훅을, napi는 watch stop() use-after-free를 처리했습니다.
