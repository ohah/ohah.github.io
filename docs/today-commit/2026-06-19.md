# 2026-06-19

## 요약 (AI 작성)

iris 쪽에서는 JSON payload 생성·라운드트립 경로의 성능을 집중적으로 손봤습니다. Uint8Array·프로퍼티·숫자·분기·object-traversal 루프에 벌크 빠른 경로를 두루 채택하고, snapshot 문자열 복사를 제거하며 compact snapshot·layout detector까지 최적화했습니다.

maru는 config를 대폭 확장한 날입니다. `theme.palette.0~15` ANSI 16색 override와 컬러 테마 프리셋(gruvbox·solarized·dracula·catppuccin 등 6종 + `theme.preset`), `font.line-height`·`letter-spacing`, 비대칭 window padding(top/right/bottom/left), `scrollback.lines`·`bell.audible` 등을 추가하고, 네이티브 메뉴에 Reload Config·Reset to Defaults를 넣었습니다. 입력 쪽은 파일/URL/텍스트 드래그앤드롭과 붙여넣기 경로 삽입, `Cmd+K` 화면+스크롤백 비우기, `Cmd+]/[` split 순환을 붙였고, 데스크톱 알림은 `TERM_PROGRAM=ghostty` 주입으로 Claude/Codex 무설정 동작과 권한 선요청을 지원합니다. 렌더링에서는 OSC 133 거터 마크를 셀 바깥으로 빼 col 0 겹침을 없애고 CAMetalLayer를 sRGB로 태그했으며, 코어 안정성 측면에서 `core_mutex` 재진입 안전망(CoreOwner)과 IME 조합 확정 시 데드락, `termLabel` use-after-free를 잡고 terminfo(xterm-maru opt-in·XTVERSION·maru ssh 전파)까지 손봤습니다.
