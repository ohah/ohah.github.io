# 2026-06-23

## 요약 (AI 작성)

maru의 유니코드 텍스트 처리(grapheme cluster)를 근본적으로 넣은 날입니다(69커밋). UAX#29 기반 한글 grapheme 분절 순수 함수(HG1)에서 출발해, 셀 다중 코드포인트 저장과 NFD 한글 cluster producer·무손실 직렬화(HG2), CoreText 셰이퍼에 cluster 전체 전달(HG3a)과 Cell.combining 제거(HG3b)로 이어졌고, emoji ZWJ 시퀀스(GB11)·키캡을 클러스터 경로로 통합했습니다. grapheme_store dedup(link_ids 패턴)으로 반복 cluster의 per-cell 증가도 차단했습니다. 이 밖에 사이드바가 접혀도 알림 종·배지를 최소 폭에 유지하도록 다듬었습니다.
