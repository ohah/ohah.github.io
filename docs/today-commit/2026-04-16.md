# 2026-04-16

2026-04-16은 Bungae와 ZTS의 Metro 호환성 강화, asset_registry 모듈 업데이트, dead code 제거 등이 이루어진 날입니다. Bungae에서는 Metro extraNodeModules와 resolveRequest에 mapper hook을 추가하여 호환성을 높이고, 빈 매핑을 disabled flag로 처리했으며, ZTS submodule 210개의 커밋을 병합했습니다.
