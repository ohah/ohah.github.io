# 2026-06-28

## 요약 (AI 작성)

인앱 업데이트와 배포 파이프라인을 갖춘 날입니다. 버전 주입·체크 로직·config 키로 새 버전 안내 토대를 만들고 app_session에 백그라운드 버전 체크와 알림 발화를 통합했으며, universal(arm64+x86_64) dmg 빌드와 릴리스 워크플로·업데이트 전략 문서까지 정리했습니다.

UI는 Warp를 벤치마킹해 상단 탭바를 다듬었습니다. grip hover-only·바 높이(1단계), 마지막 탭 옆 인라인 "+"(2단계), 실제 Warp식 lifted 캡슐 pill 탭, 사이드바 카드 여백 폴리시를 적용했습니다. 에이전트 표시는 running 회전 스피너와 색상을 넣고 Claude 색을 Anthropic 공식 코랄(#CC785C)로, 아이콘을 볼드로 바꾸는 등 사용자 피드백을 다수 반영했습니다. 이 밖에 터미널 매크로(text:/esc:/ctrl:) rhs GUI 편집, `chrome.preset` 묶음 레이아웃 프리셋, color 스와치 rich 둥근 quad 칩 렌더, config write-back pending 리스트 registry화(TDD), 모달 그림자 완화 등을 처리했습니다.
