# 2026-06-21

## 요약 (AI 작성)

스키마-주도 config 프레임워크를 세운 날입니다. 메타를 1급 필드로 두는 D안(`config-schema.md`)을 확정하고, 대표 4종 이주(CS-1)에서 시작해 sub-struct 스칼라(CS-2)와 최상위 스칼라(CS-2b)를 일괄 이주했으며, 스키마↔문서 doc-drift 가드(CS-3)와 세팅 화면(GUI) 설계(`config-gui.md`, CS-4)까지 이어 붙였습니다. 여기에 per-key write-back 일반화(`updateForKeys`), 세팅 화면 시각 배선(⌘, 열기·schema bool 행·라이브 적용)과 config 즉시 write-back을 얹었습니다.

세팅 GUI는 포인터 입력 라우팅 토대(ChromeHost.handlePointer, CS-4-0)를 깔고 text·dropdown(enum)·slider 위젯과 좌측 Section 그룹 네비를 붙였습니다. SSH 통합은 별도 전략 문서를 신설하고, control socket 결정론적 경로(1단계)·OSC 5379 원격 세션 인식(2단계)·클립보드 이미지 paste 업로드(4단계)와 JPEG 지원·PID 충돌 방지·7일 보존까지 이미지 드롭 흐름을 완성했습니다. 이 밖에 `shell.command/args` 커스텀 셸, alt screen에서의 ⌘F Find, 확인 모달 마우스 hit-test, 탭 전환 시 IME 조합 누수, split 탭 새 워크스페이스 크기 물림 버그 등을 처리했습니다.
