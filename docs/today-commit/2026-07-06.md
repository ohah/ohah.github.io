# 2026-07-06

## 요약 (AI 작성)

사이드바 그룹 고정 계열을 한 축 더 넓혀 그룹-로컬 pin(GL1~4)을 추가했습니다. `local_pinned` 필드·직렬화와 `stablePartitionSubtree` 토대(GL1) 위에 `toggleLocalPin`과 멤버 우클릭·드래그 clamp(GL2), 📌 렌더·위생·마커 위 배치(GL3), 로컬 pin 공존·중첩 하드닝(GL4)을 얹었습니다. 이 축은 그룹 고정(C2)과 직교하는 새 개념이라 §13 문서에 적대적 검증을 세 차례 반영했고, 최상위 카드가 그룹에 흡수되거나 그룹 고정 해제 시 로컬 pin이 잔류하던 버그를 pin 회귀 매트릭스로 함께 잡았습니다. 그룹 드래그도 Cmd=중첩·일반=형제로 구분하고 고스트 피드백을 붙였습니다.

문서 쪽은 세션 컨트롤 플레인 설계가 큰 비중을 차지했습니다. JSON-RPC 2.0 + ndjson framing 프로토콜, capability fd 인가 계획, 네이티브 뷰 예외 명문화와 Phase별 테스트 전략을 세우고 적대적 검증을 여러 렌즈로 반영했습니다. web-panel 쪽도 WKWebView 합성·z-order·브리지 인프라, CEF opt-in 플러그인 방향, `maru-app://` 스킴(RFC 3986 근거)을 정리했습니다. PR로는 이 설계가 실제 코드로 내려와, 컨트롤 소켓 부트스트랩과 Surface DTO·read-only 디스패치, CLI `sessions list/get`, capability auth 코어(1a~1e)와 `SurfaceIdAllocator`·`WindowMembershipSnapshot`·`WindowGraph`·`LiveSurfaceRegistry`(M0a~M2a)가 차례로 머지됐습니다. 그 사이 OSC 52 대용량 클립보드 유실·회수 갭 수정, ANSI 팔레트 자동 대비 게이트로 라이트 테마 가독성 개선도 이뤄졌습니다.
