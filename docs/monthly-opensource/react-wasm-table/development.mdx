## 핵심 아키텍처

```
React Headless API → JS Adapter → WASM Core (Rust/Taffy) → Canvas Renderer
```

이 4-layer 파이프라인이 프로젝트의 핵심이다. 각 레이어의 설계와 구현 과정은 하위 문서에서 다룬다.
