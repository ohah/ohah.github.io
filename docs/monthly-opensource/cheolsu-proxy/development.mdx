## 기능 전체 목록

### 프록시 코어
- HTTP/HTTPS 트래픽 인터셉션
- WebSocket 캡처 및 메시지 주입
- TLS 1.0/1.1 레거시 클라이언트 지원 (Hybrid TLS Handler)
- 업스트림 프록시 지원 (HTTP/HTTPS/SOCKS + 인증)
- 네트워크 스로틀링 (대역폭 제한)
- 비동기 압축 해제 (Brotli, Gzip, Zlib, Zstd)

### 트래픽 조작
- **Intercept Rules** — Block, Modify Request, Modify Response, Map Local, Map Remote
- **Server Replay** — 캡처된 응답 자동 재생
- **Request Replay** — 개별/연속 요청 재전송
- **Scripting** — TypeScript 기반 요청/응답/WebSocket 조작 (Deno Core + V8)

### 필터링 및 내보내기
- **Cheolsu-Query** — 전용 쿼리 언어 (메서드, 상태, URL + 논리 연산자)
- **HAR Export** — HTTP Archive 형식 내보내기

### 인터페이스
- GUI 데스크톱 앱 (Tauri + React)
- TUI 터미널 인터페이스 (Ratatui)
- MCP 서버 (14개 도구)
- 다크/라이트 테마 (5종)
- i18n (영어, 한국어)

### MCP 도구 목록 (14개)

**트래픽 검사**: `search_traffic`, `get_transaction`, `get_websocket_messages`

**요청 전송**: `replay_request`

**규칙 관리**: `list_rules`, `add_rule`, `remove_rule`

**프록시 상태**: `proxy_status`, `clear_traffic`
