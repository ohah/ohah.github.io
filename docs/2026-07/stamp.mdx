# 🌈 Magic Rainbow

## Intro to CSS Gradients (Part 1)

In this post, we'll explore the beautiful world of **CSS gradients**, specifically focusing on linear and radial ones. By mastering these techniques you can create stunning visual effects with minimal code.

### Linear Gradient Syntax Example: `background-image`

The basic syntax for a liner gradient is:

```css
linear-gradient(direction or angle,
  color1, start-position %?
)
```
